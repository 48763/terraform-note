# Terraform-note

Terraform 是由 HashiCorp 創建的開源基礎架構即代碼軟件工具。運維人員使用 HCL（HashiCorp Configuration Language）定義資料中心的基礎設施。

## 前言

雲端服務使用 *google cloud platform*，版控存儲使用 *github*，運作系統使用 *ubuntu*。
